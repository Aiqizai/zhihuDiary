# zhihudiary

## 技术栈
vue-cli + vue-router + ES6 + axios + sass + vant UI框架

## 功能
+ 当天的日报以及热门日报
+ 滚动刷新加载前一天的日报
+ 登录
+ 查看日报详情
+ 查看对应日报的评论，包括长评和短评
+ 搜索日报
+ 收藏和取消收藏日报

## 项目运行

```   
git clone https://github.com/Aiqizai/zhihuDiary.git
cd zhihuDiary
npm install

npm run serve

### Compiles and minifies for production
